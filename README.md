# Banana Scan
a spectrometer-based fruit ripening detector, built as a final project for ECE 4760 (microcontrollers). final website available [here](http://people.ece.cornell.edu/land/courses/ece4760/FinalProjects/f2018/cc2294_mf568_rms438/cc2294_mf568_rms438/cc2294_mf568_rms438/index.html). 

main code for the project can be found in the /code section. The microcontroller is the Microchip PIC32 series programmed with MPLABX in C.
